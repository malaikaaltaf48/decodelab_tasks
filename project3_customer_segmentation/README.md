# Project 3 — Customer Segmentation

**Dataset:** Mall Customers Dataset (loaded via raw GitHub URL for full reproducibility)

## Overview
This project applies unsupervised learning to segment mall customers into distinct groups based on spending behavior and demographics.

## Key Steps
- Data preprocessing using StandardScaler and LabelEncoder
- Dimensionality reduction with PCA
- Optimal cluster selection using the Elbow Method and KneeLocator
- Cluster quality evaluation using Silhouette Score
- Centroid reverse-engineering to derive business personas
- Final results exported to CSV
